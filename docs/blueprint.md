# **App Name**: NEU ScholarFlow

## Core Features:

- Secure Visitor Check-in: Allow visitors to securely check in via simulated RF ID scan or institutional Google email. Upon successful authentication, display the visitor's name, college/office, and a 'Welcome to NEU Library!' message.
- Purpose of Visit Input: Provide a user interface for visitors to select their primary purpose for visiting from a pre-defined list (e.g., Reading Books, Research, Computer Use, Assignments) upon entry.
- Admin Visitor Management: Enable administrators to view current and past visitor logs and block specific visitors from entry through a dedicated interface.
- Admin Statistics Dashboard: Present administrators with a dynamic dashboard displaying real-time and historical visitor statistics per day, week, month, or custom duration ranges.
- AI-Powered Trend Summaries: Utilize a generative AI tool to analyze visitor data and provide administrators with natural language summaries of key trends, peak usage times, and any identified anomalies.
- Exportable Statistical Reports: Allow administrators to generate and download comprehensive statistical reports in PDF format based on selected date ranges and filters from the dashboard.

## Style Guidelines:

- Color Palette: Drawing inspiration from the concepts of 'Knowledge', 'Focus', and 'Modern Learning Environment', a light color scheme is chosen to reflect a bright, conducive learning space. The primary color, a deep, muted blue (#264D73), signifies trust and intellect. The background color, a heavily desaturated blue-gray (#F1F5F8), offers a clean, serene backdrop, while the accent color, a vibrant aquamarine (#36BBDB), provides a refreshing contrast and draws attention to interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly technical yet legible feel, suitable for titles and important alerts. Body text font: 'Inter' (sans-serif) for its neutral, highly readable qualities, ensuring clarity across data tables, reports, and longer text passages.
- Employ a consistent set of clean, vector-based icons with simple outlines to maintain clarity and usability. Icons should visually represent functions such as check-in, administration, reports, and statistics without clutter.
- The layout should be intuitive and organized. The visitor check-in interface will feature a minimalist, focused design to streamline entry. The admin dashboard will use a modular grid system to clearly present statistics and management tools, ensuring easy navigation and information retrieval.
- Subtle, non-disruptive animations will be used for interactive feedback, such as successful check-in confirmation and transitions between dashboard views, enhancing the user experience without causing distractions.
