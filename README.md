# New Era INC Library - Visitor Log System

A secure, high-performance visitor check-in and administration system for the New Era INC Library.

## Features
- **Visitor Terminal**: Secure check-in for registered students with real-time verification and block-list checks.
- **Admin Console**: Comprehensive dashboard for library administrators.
  - Real-time visitor statistics and visual analytics.
  - Detailed audit logs with filtering and search.
  - PDF Report generation for Today, Last Week, and Last Month.
  - Visitor registry management (registration and administrative blocking).
- **Security**: Robust Firebase Security Rules ensuring data integrity and privacy.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Database/Auth**: Firebase Firestore & Authentication
- **Styling**: Tailwind CSS & ShadCN UI
- **PDF Generation**: jsPDF & AutoTable
- **Icons**: Lucide React
